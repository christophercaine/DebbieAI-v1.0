EXECUTIVE SUMMARY
Caine is building DebbieAI - a comprehensive mobile application suite designed to solve the biggest pain points for home repair contractors and small construction businesses.
The goal is NOT to build another generic project management tool. The goal is to build the first AI-powered assistant specifically designed for contractors that turns chaos into efficiency and photos into profits.

THE PROBLEM CAINE IS SOLVING
The Contractor's Daily Nightmare
Meet Bob - a typical home repair contractor:
📸 Photo Chaos: Takes 80+ photos per job site. They're scattered across his phone, cloud, USB drives. Can't find "that bathroom photo from the Thompson job" when the client calls.


📋 Lost Customer Data: Phone numbers and emails scattered across his phone, text messages, business cards, napkins. Calls the wrong number. Misses follow-ups.


⏰ Time-Sink Estimates: Spends 3-4 hours measuring, calculating, creating estimates. Makes calculation mistakes = loses money. No consistency.


📞 Forgotten Follow-ups: Doesn't remember when to call people back. Misses re-contact opportunities. Lost sales.


🤷 No Business Intelligence: Has no idea which customers are most profitable. Doesn't know seasonal trends. Can't optimize his business.


Result: Bob works 60+ hour weeks and feels disorganized despite being competent at his trade.
The Gap in the Market
Existing solutions:
HubSpot: Generic CRM, too expensive ($1000+/month), not construction-focused
Buildr/BuilderTREND: Project management, but doesn't handle photo enrichment, costs $200+/month
ServiceTitan: Focused on plumbing/HVAC, not general contractors
Houzz Pro: Designer-focused, not contractor-focused
Procore: Enterprise-only, overkill for 1-10 person teams
What's missing: There is NO product that does all five things Bob needs in one integrated suite at an affordable price ($50/month or less).

### Code: `aiRouter.ts`
```typescript
import { GoogleGenerativeAI } from "@google/generative-ai";
import Anthropic from "@anthropic-sdk/sdk";
import { Mistral } from "@mistralai/mistralai";
import Replicate from "replicate";
import redis from "redis";

type AITask = 
  | "photo_analysis"
  | "estimate_generation"
  | "bulk_tagging"
  | "voice_command";

interface AIRequest {
  task: AITask;
  input: string;
  images?: string[];  // base64
  context?: Record<string, any>;
}

interface AIResponse {
  success: boolean;
  provider: string;
  result: string;
  cost?: number;
  error?: string;
}

export class AIRouter {
  private gemini: GoogleGenerativeAI;
  private claude: Anthropic;
  private mistral: Mistral;
  private replicate: Replicate;
  private redis: redis.RedisClient;
  private ollamaUrl: string;

  constructor() {
    this.gemini = new GoogleGenerativeAI(process.env.GEMINI_API_KEY || "");
    this.claude = new Anthropic({
      apiKey: process.env.ANTHROPIC_API_KEY || "",
    });
    this.mistral = new Mistral({
      apiKey: process.env.MISTRAL_API_KEY || "",
    });
    this.replicate = new Replicate({
      auth: process.env.REPLICATE_API_TOKEN || "",
    });
    this.redis = redis.createClient({
      url: process.env.REDIS_URL || "redis://localhost:6379",
    });
    this.ollamaUrl = process.env.OLLAMA_URL || "http://localhost:11434";
  }

  /**
   * Main router: route request to best provider
   */
  async process(request: AIRequest): Promise<AIResponse> {
    console.log(`[AIRouter] Processing task: ${request.task}`);

    // Try cache first
    const cacheKey = this.getCacheKey(request);
    const cached = await this.getFromCache(cacheKey);
    if (cached) {
      console.log(`[Cache] HIT for ${request.task}`);
      return JSON.parse(cached);
    }

    let response: AIResponse;

    switch (request.task) {
      case "photo_analysis":
        response = await this.handlePhotoAnalysis(request);
        break;

      case "estimate_generation":
        response = await this.handleEstimateGeneration(request);
        break;

      case "bulk_tagging":
        response = await this.handleBulkTagging(request);
        break;

      case "voice_command":
        response = await this.handleVoiceCommand(request);
        break;

      default:
        response = {
          success: false,
          provider: "unknown",
          result: "",
          error: "Unknown task",
        };
    }

    // Cache successful responses for 24 hours
    if (response.success) {
      await this.setCache(cacheKey, JSON.stringify(response), 86400);
    }

    return response;
  }

  /**
   * PHOTO ANALYSIS: Google Vision (Gemini) → Replicate → Ollama
   */
  private async handlePhotoAnalysis(req: AIRequest): Promise<AIResponse> {
    // Try 1: Google Vision with Gemini
    try {

  /**
   * OLLAMA FALLBACK: Always available, free
   */
  private async fallbackToOllama(req: AIRequest): Promise<AIResponse> {
    try {
      const response = await fetch(`${this.ollamaUrl}/api/generate`, {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({
          model: "llama2",
          prompt: `${req.input}\n\nRespond in JSON format.`,
          stream: false,
        }),
      });

      const data = (await response.json()) as { response: string };

      return {
        success: true,
        provider: "ollama",
        result: data.response,
        cost: 0,
      };
    } catch (error) {
      return {
        success: false,
        provider: "ollama",
        result: "",
        error: `Ollama unavailable at ${this.ollamaUrl}`,
      };
    }
  }

  // ===== CACHE HELPERS =====

  private getCacheKey(req: AIRequest): string {
    const hash = Buffer.from(
      req.task + "|" + req.input + "|" + (req.images?.[0]?.substring(0, 50) || "")
    ).toString("base64");
    return `ai:${req.task}:${hash}`;
  }

  private async getFromCache(key: string): Promise<string | null> {
    try {
      return await this.redis.get(key);
    } catch {
      return null;
    }
  }

  private async setCache(
    key: string,
    value: string,
    ttl: number
  ): Promise<void> {
    try {
      await this.redis.setEx(key, ttl, value);
    } catch {
      // Silently fail cache
    }
  }
}
```

## TROUBLESHOOTING
*   **"Ollama not found"**
    *   Solution: `ollama pull llama2 && ollama serve`
    *   Then check `http://localhost:11434/api/tags`
*   **"API key invalid"**
    *   Solution: Check `.env` file exists and has correct keys
    *   Run: `echo $GEMINI_API_KEY` (should print key)
*   **"Redis connection failed"**
    *   Solution: Start Redis: `redis-server`
    *   Or use cloud Redis from Render/Railway
*   **"Costs too high"**
    *   Solution: Check `AI_TRACK_COSTS=true` logs
    *   Review which provider is used most
    *   Adjust fallback chain to prefer cheaper APIs
